해봐야할 예제

1️⃣ 여러 개의 드래그 가능한 아이템 추가하기

id: "draggable-1", "draggable-2" 등으로 여러 개의 Draggable 컴포넌트를 만들어 본다.
여러 개의 아이템이 독립적으로 드래그될 수 있도록 구조를 설계한다.
2️⃣ 드롭 가능한 영역 추가하기 (useDroppable)

useDroppable을 사용하여 특정 박스 안에서만 드롭되도록 만들어 본다.
여러 개의 드롭존을 만들어 드래그한 요소가 특정 영역에만 놓이도록 구현한다.
3️⃣ 리스트 정렬 기능 만들기 (SortableContext, useSortable)

SortableContext, useSortable을 사용하여 리스트 내에서 드래그 앤 드롭을 통한 정렬이 가능하도록 구현한다.
리스트에서 아이템을 드래그하여 순서를 변경할 수 있도록 한다.
4️⃣ 드래그 시 스타일 변경하기 (isDragging 상태 관리)

isDragging 상태를 만들어 드래그 중일 때 opacity를 조절한다.
cursor: grabbing, scale 효과 등을 추가하여 드래그 중임을 직관적으로 표시한다.
5️⃣ 정리 및 실전 응용

여러 개의 드래그 가능한 아이템을 특정 영역으로 옮기고, 리스트 내에서 정렬이 가능하도록 결합해본다.
복잡한 UI에서 실제로 사용할 수 있도록 구조를 개선하고 최적화한다.
